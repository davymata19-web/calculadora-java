# Calculadora em Java

## Descrição
Sistema de calculadora via terminal que permite ao usuário realizar operações matemáticas básicas de forma contínua até encerrar o programa.

## Tecnologias
- Java

## Funcionalidades
- Operações: +, -, *, /
- Entrada dinâmica de números
- Encerramento com '='

## Aprendizados
- Estruturas de repetição (while)
- Condicionais (if/else)
- Entrada de dados com Scanner
